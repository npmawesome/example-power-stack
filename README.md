example-power-stack
===================

> This example refers to the article on [strongloop.com][1] and reposted on [npmawesome.com][2].

## Usage

Clone the example:

    git clone https://github.com/npmawesome/example-power-stack.git

Install dependencies:

    npm install

Run the test:

    npm test

Profit!
